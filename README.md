# 🔍 Naukri Smart Filters

> **Filter smarter. Apply better.**  
> A Chrome extension that adds powerful custom filters to Naukri job search — salary ranges, keyword matching, company blacklists, and more.

---

## 🚀 Installation

### Manual Installation (Developer Mode)
1. Clone this repository
   ```bash
   git clone [https://github.com/yourusername/naukri-smart-filters.git](https://github.com/professor91/naukri-extension)
   ```
2. Open Chrome and go to `chrome://extensions/`
3. Enable **Developer mode** (toggle in top-right)
4. Click **Load unpacked** and select the cloned folder
5. The extension icon will appear in your toolbar
